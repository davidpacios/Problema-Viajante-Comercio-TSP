P4.1: Implementacin de Sistemas de Colonias de Hormigas (SCH)

Implementa el algoritmo de sistemas de colonia de hormigas que materializa la MH vista en la sesin expositiva y revisa las notas particulares para el problema de VC. Para facilitar vuestra labor, se os proporciona aqu una descripcin algortmica de referencia.

1. Establecer parmetros: $G, m, q_{0}, \tau_{0}, \rho, \xi, iter_{min}, iter_{max}, iter_{sinmejoras}, ratio_{mejora}$

2. Inicializar pistas de feromonas: $\forall (i, j) \tau_{ij} = \tau_{0}$

3. HACER 3.1. PARA cada hormiga $k$ en colonia m HACER

3.1.1.construir una solucin para $k$ siguiendo la regla de decisin a cada paso: $$ j=\begin{cases}argmax_{l\in N_i^k} \{ [\tau_{il}]^\alpha [\eta_{il}]^\beta \}, \text{if } q \leq q_0; \\ J, \text{en otro caso;}\end{cases} $$, donde $J$ es la exploracin dirigida por:

$$ p_{ij}^{k} = \frac{[\tau_{ij}]^\alpha [\eta_{ij}]^\beta}{\sum_{l\in N_i^k} [\tau_{il}]^\alpha [\eta_{il}]^\beta}, \text{if } j\in N_i^k $$

3.1.2. (opcional) Aplicar actualizacin de feromona local/online con la regla: $$ \tau_{ij} \leftarrow (1 - \xi) \tau_{ij} + \xi \tau_{0}$$ 3.2. FIN PARA

3.3. Evala soluciones en la colonia m y registrar la mejor solucin hasta el momento $T^{bs}$

3.4. Aplicar actualizacin de feromona global/offline con la regla: $$\tau_{ij} \leftarrow (1-\rho) \tau_{ij} + \rho \Delta\tau_{ij}^{bs}, \forall(i, j) \in T^{bs}$$

4. HASTA cumplir la condicin de parada

La implementacin debe permitir la inicializacin de los parmetros y estructuras de informacin asociadas como sigue:

  • G es la estructura de grafo que permite calcular distancias entre ciudades sobre una instancia del problema dada (e.g., fichero de localizaciones).
  • Probabilidad de explotacin $q_{0}=0.5$.
  • Inicializacin de la informacin heurstica: $\eta_{ij}=1/d_{ij}, \forall (i, j)$, donde $d_{ij}$ es la distancia directa entre las ciudades i y j.
  • Inicializacin de las pistas de feromonas (depsito inicial): $\tau_{ij}= \tau_{0}= 1 /(n C^{NN}) \forall (i, j)$ donde n es el nmero de ciudades y $C^{NN}$ es el coste del tour generado siguiendo una estrategia voraz del vecino ms prximo (NN, Nearest Neighbour).
  • Parmetros de influencia de las pistas de feromonas ($\alpha$) e informacin heurstica ($\beta$) en la regla de decisin establecidos a 1 y 3 respectivamente
  • Ratio de evaporacin de feromona: debers probar con diferentes magnitudes (p.ej., $\rho=0.01$, $\rho=0.1$ y $\rho=0.5$)
  • Actualizacin de feromona global con depsito elitista basado en la mejor solucin (bs, best solution): $\tau_{ij} \leftarrow (1-\rho) \tau_{ij} + \rho \Delta\tau_{ij}^{bs}, \forall (i, j) \in T^{bs}, donde \Delta\tau_{ij}^{bs}=1/C^{bs}$
  • Actualizacin de feromona local siguiendo la regla: $\tau_{ij} \leftarrow (1- \xi) \tau_{ij} + \xi \tau_{0}, con \xi=0.1$
  • Condicin de parada que de manera combinada tenga en cuenta $iter_{min}$, $iter_{max}$, $iter_{sinmejoras}$, $ratio_{mejora}$
  • La implementacin debe ser completamente parametrizable, de forma que todos los componentes susceptibles de ser ajustados mediante parmetros puedan ser establecidos en cada ejecucin/problema, y que puedas modificar la inicializacin heurstica y de pistas de feromona, puedas fcilmente reemplazar las reglas de actualizacin de feromonas, activar/desactivar la actualizacin local de actualizacin de feromona, etc.

def solucion_voraz(Loc):
        solucion = [0]
        for i in range(1, Loc.nciudades):
            mejor_punto, mejor_coste = None, None
            
            for nueva_ciudad in range(1, Loc.nciudades):    
                if nueva_ciudad in solucion:
                    continue
                    
                coste_nueva_ciudad = Loc.distancia(solucion[len(solucion) - 1], nueva_ciudad)
                
                if not mejor_punto or coste_nueva_ciudad < mejor_coste:
                    mejor_punto, mejor_coste = nueva_ciudad, coste_nueva_ciudad
            
            solucion.append(mejor_punto)
            
        solucion = solucion  + [0]    
        return solucion

def inicializar_Nij(Loc):
    n = [[0.0 for _ in range(Loc.nciudades)] for _ in range(Loc.nciudades)]
    for i in range(Loc.nciudades):
        for j in range(i + 1, Loc.nciudades):
            n[i][j] = 1 / Loc.distancia(i, j)
            n[j][i] = 1 / Loc.distancia(j, i)
    return n

def inicializar_Tij(Loc):
    cost = coste(Loc, solucion_voraz(Loc))
    aux = 1 / (cost * Loc.nciudades)
    return [[aux for _ in range(Loc.nciudades)] for _ in range(Loc.nciudades)]

def calcular_soluciones_hormigas(Loc, m, q0, t, n, alpha, beta):
    soluciones = []
    
    for i in range(m):
        solucion = [0]
        
        ciudades_sin_visitar = list(range(1, Loc.nciudades))

        while len(ciudades_sin_visitar) > 0:
            
            ultima_ciudad_visitada = solucion[len(solucion) - 1]
            q = random.uniform(0, 1)
            
            if q < q0:
                siguiente_ciudad = explotacion(ciudades_sin_visitar, ultima_ciudad_visitada, t, n, alpha, beta)
            else:
                siguiente_ciudad = exploracion_dirigida(ciudades_sin_visitar, ultima_ciudad_visitada, t, n, alpha, beta)

            solucion.append(siguiente_ciudad)
         
            ciudades_sin_visitar.remove(siguiente_ciudad)

        solucion = solucion + [0]
            
        soluciones.append((solucion, coste(Loc, solucion)))

    return soluciones

import statistics
import time
from tqdm.notebook import trange, tqdm
def pruebas(nombre_prueba, algoritmo, Loc, m, q0, alpha, beta, p, iter_min, iter_max, iter_sin_mejora_limite, ratio_mejora_limite):
    tiempos = []
    resultados = []
    print(nombre_prueba)
    
    for i in tqdm(range(5), desc=nombre_prueba):
        inicio = time.time() 
        a = algoritmo(Loc, m, q0, alpha, beta, p, iter_min, iter_max, iter_sin_mejora_limite, ratio_mejora_limite)
        delta_time = time.time() - inicio
        tiempos.append(delta_time)
        resultados.append(a[1])

    mediaT = statistics.mean(tiempos)
    print("La media de los tiempos es: ", mediaT)
    desvtipT = statistics.stdev(tiempos)
    print("La desviacion de los tiempos es: ", desvtipT)
    
    mediaR = statistics.mean(resultados)
    print("La media de los resultados es: ", mediaR)
    desvtipR = statistics.stdev(resultados)
    print("La desviacion de los resultados es: ", desvtipR)
    print("-------------\n")
    
    return (resultados, tiempos, mediaR, mediaT)
